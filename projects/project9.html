Autonomous Indoor Navigation Robot

SIANV is an indoor autonomous robot developed as a final-year project (2024–25). It navigates indoor environments using path planning and obstacle avoidance, relying on a 2D LiDAR for LaserScan and odometry via scan matching. We used Google Cartographer SLAM for mapping and localization. AMCL was initially tested but removed due to the single-sensor setup, with Cartographer providing sufficient localization. The goal was to convert an industrial electric vehicle into an autonomous robot with minimal hardware changes.

We used ROS 2 with the Nav2 stack, employing a hybrid A* global planner and a local planner for real-time obstacle avoidance. Custom ROS 2 nodes handled sensor input, motor control, and state publishing. We also implemented a custom reactive planner for unstructured navigation and a voice-controlled mode. The robot was tested in a lab and home setup, showing reliable autonomous movement. Built in 1–2 months, this project greatly enhanced my understanding of SLAM, localization, and system integration.

Reactive Planner (Non Pre-Built Map Technique)

The reactive planner uses proportional control based on the target’s direction and distance. It considers a field of view (FOV) split into left and right halves. If an obstacle is detected in one half, the robot turns through the other. When both halves have obstacles, the planner computes bypass nodes and chooses the one that minimizes distance to the target. It dynamically re-evaluates and continues navigation using the same approach.

Voice-Controlled Mode

Voice control enables users to interact with the robot through speech. Commands like “go forward” or “turn left” are processed via speech recognition and mapped to navigation commands. This provides an intuitive way to control the robot without physical input devices. The feature combines real-time speech-to-text conversion with command parsing and robot actuation.
