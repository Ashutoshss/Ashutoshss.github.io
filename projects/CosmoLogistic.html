Cosmo Logistic

e-Yantra, IIT BOMBAY:

I participated in the e-Yantra competition by IIT Bombay, where I worked on the topic "Cosmo Logistics," one of several problem statements. During the project, I developed an algorithm for autonomous warehouse operations, involving mobile robot navigation and robotic arm manipulation for picking and placing packages. Implemented in ROS 2, the project included mapping the warehouse using SLAM techniques, navigating the robot with the Navigation Stack 2, and performing arm manipulation using MoveIt 2.

The task was divided into three parts, one of which was object detection. The packages contained ArUco markers, and my algorithm's function was to collect raw image data and depth images, then process them to detect the markers and estimate their position and orientation. This involved preprocessing the raw images to reduce noise, detecting edges, identifying marker IDs, and calculating the center, orientation, depth, and other relevant parameters.

The other task involved developing a program for picking and placing packages based on their respective IDs. We explored multiple approaches to perform the manipulation:

Direct Joint Control – In this method, we directly provided joint positions to manipulate the arm and used an electromagnet placed at the end-effector (EEF) to attach the package.

MoveIt-Based Control – This approach involved detecting the package’s orientation and position, then sending this data to the pymoveit2 package. MoveIt computed the inverse kinematics and executed the manipulation. This method also involved adding objects to the Planning Scene to ensure collision avoidance, which is essential since robotic manipulators are typically used to interact with their environment safely.

Servoing Techniques – In this approach, we controlled the robot arm’s axes using velocity commands. We also implemented a proportional (P) control to enhance the arm’s speed and automate the process.

It was a great experience working on all the possible techniques to complete the arm manipulation task. will definately work on it future projects.

We also implemented a program to navigate the robot in the warehouse using the Navigation Stack 2. This involved mapping the warehouse using SLAM techniques, where we used slam_toolbox, which allowed us to create a map of the environment and localize the robot using AMCL. The Navigation Stack 2 was then used to plan paths and navigate the robot to specific goal positions within the warehouse.

I implemented it on both simulation and actual hardware through remote access. Although we were not able to win the competition, we successfully implemented the algorithm on the project, and it was a great learning experience.

We also completed the hardware tasks but did not score enough to reach the finals. Additionally, I have very limited data on the projects mostly simulation because I didn’t thoroughly document them.
